# GRAPH × GRAPH リリースノート

---

## v3.0.0

### UI 全体の刷新

- カラースキームをインディゴ系に統一し、CSS カスタムプロパティで管理
- コンテナをレスポンシブ対応（固定幅 1100px → `max-width: 1100px`）
- ナビゲーションバーをブランドカラー + ドロップシャドウでモダンに
- バージョン表示をピルバッジ化（位置ハック除去）
- スムーズトグルをカード UI から独立したトグルエリアに変更
- ボタン・入力欄のホバー・フォーカスエフェクトを整備

### リアルタイム可視化

- 入力テキストの変更を 500ms デバウンスで自動的にグラフに反映
- VISUALIZE!! ボタンを押さなくても即座にプレビューが更新される

### サンプルプリセット

- ワンクリックでサンプルグラフを読み込めるプリセットボタンを追加
- クリックのたびにランダム生成（頂点数・辺の構成が毎回変わる）
- 対応プリセット: スター / パス / サイクル / 完全グラフ / 二部グラフ / 重み付き木 / 重み付きグラフ / DAG

### グラフ情報パネル

可視化後に以下の情報を自動表示:

- 頂点数・辺数
- 連結成分数（無向グラフ）/ SCC 数（有向グラフ）
- グラフ種別バッジ（木 / 森 / 完全グラフ / 二部グラフ / DAG / 強連結）

### 最短経路表示

- 始点・終点を入力してワンクリックで最短経路を可視化
- 重みなし: BFS / 重みあり: Dijkstra で自動切り替え
- 始点（青）・終点（赤）・経路上のノード（オレンジ）を色分けハイライト
- 経路パネルにノード列を表示（例: 1 → 3 → 5）
- 経路なしの場合も明示

### トポロジカル順序

- DAG を入力した際にトポロジカル順序を自動計算・表示（Kahn's algorithm）

### 辺数省略の自動補完

- 木に限らず任意のグラフで辺数 M を省略した入力に対応
- 重み付き・なしを問わず、辺の幅から M を自動推定して補完

### URL Export

- 現在のグラフ設定（入力・形式・有向/無向・重み付き）を URL にエンコードしてクリップボードにコピー
- URL を共有するだけで同じグラフを再現可能
